DAB AI frontend (Next.js) for the DAB AI marketing agent platform.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open the dev URL printed by Next.js in your terminal.

To point the frontend at the production API, set `NEXT_PUBLIC_API_BASE` to your API base URL (example: `https://api.dabcloud.in/api`).

Production URLs:
- Frontend: `https://dabcloud.in` (or `https://app.dabcloud.in`)
- Backend: `https://api.dabcloud.in`
